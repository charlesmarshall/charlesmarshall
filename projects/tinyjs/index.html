Tiny JS

A long time ago I created a very handy little site called tiny js ; it was all about small but powerful javascript plugins for libraries such as jQuery.

At one point it made it in to an article on Smashing Magazine regarding javascript tools, resources etc.

I simply haven't had the time to sort out the niggling bugs (git syncing for example) or give it the revamp it needed. The plans were grand, but never realised and I've now closed the site down

All of the plugins are still available on the github page.
