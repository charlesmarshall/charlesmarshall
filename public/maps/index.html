Example

During a recent project I created several google map javascript classes to my life simpler ranging from a basic map which just shows the location to a full on monster with searching, directions and filters.

Here I'm using the search version which has everything.

I also created 99 numbered map markers for anyone thats interested they are here.

If you have any questions or want to download everything you can find this project over on git hub.

  • Marker icons
  • Basic map class
  • Filter map class - certain layers can be turned on or off - uses kml (should really write an example for this)
  • Directions map class - extends the above - has tabbed bubbles and this time runs off a json file
  • Search map class - extends the above - allows inline searching with smart map repositioning
  • All map class - All in one file
  • All map class - All in one file - minimised
