Ooo, the very cool "github":http://github.com that handles all your GiT repository needs has now introduced 'pages'. This can be an entire repo or a single branch (called gh-pages).

It uses Jekyll and renders textile, markdown or good old html. For more details you can have a look at "the official page":http://pages.github.com/ or a "really nice blog post":http://blog.envylabs.com/2009/08/publishing-a-blog-with-github-pages-and-jekyll/ I found talking about how to use them as a blog
